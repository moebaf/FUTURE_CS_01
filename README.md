# FUTURE_CS_01

# Vulnerability Assessment Report

## Introduction

For this assessment, I conducted a read-only vulnerability assessment of the OWASP Juice Shop application using passive security testing techniques. The objective was to identify common security weaknesses, classify the associated risks, and recommend practical remediation measures while following ethical security testing practices.
